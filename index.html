About this site

This is just a way to catalogue my ever growing collection of mechanical keyboards with some noteworthy information regarding each. There is no proper order below, but I try to keep newest first.

Keyboards

Use dropdown menu above for quick selection, or check catalogue below to find the different keyboards currently built. Most of the builds have additional separate pages, except the last few ones.

Short info

All my keyboards are built with ISO-SE layout except for the ortho boards that has a self-composed layout. You can also check out my Instagram for more updates and information. Contact me if you have any questions.
